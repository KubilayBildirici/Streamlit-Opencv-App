## Opencv installation

Make sure the  OpenCV version (3.x or 4.x) is installed on your system

You should write the following line of code in the terminal
```
pip install opencv
```

import opencv

```rust
import cv2
```
You should write "cv2" not "opencv"
## Streamlit Run
You should write the following line of code in the terminal
```streamlit
streamlit run main_run.py
```
